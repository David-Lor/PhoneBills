# PhoneBills
PhoneBills script for Grand Theft Auto V

With PhoneBills calls in GTA V won't be free anymore. Now you will have to pay when you call somebody with your cell phone. This is quite basic and maybe doesn't make the game more harder, but you might like it if you want to pay for more things on the game, making it a bit more reallistic.
